We believe developers and researchers deserve better tools, so that they can focus on building new exciting things. That is why we have made Juno, a Jupyter/IPython notebook client for iPad. Juno lets you connect to a notebook server and do everything you do on desktop, anywhere.

Built for tablet

Built for tablet

Optimized for iPad and iOS, benefitting from responsive and tablet-friendly interface with smooth scrolling, Juno delivers a full-featured code editor experience and a complete development environment on your iPad.

Hardware keyboard support

Hardware keyboard support

Connect your iPad Pro Smart Keyboard or other Bluetooth keyboard compatible with iPad, and use all familiar Jupyter and IPython keyboard shortcuts.

Multitasking

Multitasking

If you have an iPad Air 2 or later, you can use Juno in iOS Split-screen mode alongside other apps. Run notebooks with documentation, terminal, or other editor in front of you.

Extended on-screen keyboard

Extended on-screen keyboard

Use extended on-screen keyboard designed specifically for Python and Jupyter to easily insert common special characters and perform quick actions on cells.

Code completion

Code completion

Get code completion suggestions for your programming language as you type, powered by your Jupyter notebook server's kernel.
