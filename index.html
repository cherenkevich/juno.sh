Jupyter for iOS

Juno
Juno is a complete Jupyter development environment for your iPad or iPhone — run notebooks locally on your iOS device using embedded Python kernel and integrated libraries, such as NumPy, Matplotlib and Pandas.
Download on the Apple Store

Integrated into iOS

Open notebooks from anywhere in iOS with a single tap. Whenever you edit a notebook, all your changes will be saved in the original location — even when opened from another app!

Ships with powerful tools

Juno brings NumPy, Matplotlib and Pandas to iOS — and you can easily install other pure Python packages listed in PyPI (Python Package Index) with Juno's package installer.

Full-featured notebook editor

Juno's notebook editor is fully optimized for iOS, providing hardware keyboard support and extended on-screen keyboard — as well as other Jupyter features, such as code completion.
Download on the Apple Store
