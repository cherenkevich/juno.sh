We believe developers and researchers deserve better tools, so that they can focus on building new exciting things. That is why we have made Juno, a Jupyter Notebook client for iPad. Juno lets you connect to a notebook server and do everything you do on desktop, anywhere.

Built for tablet

Built for tablet

Optimized for iPad, so that you can use your notebooks via responsive and tablet-friendly interface with smooth scrolling.

Hardware keyboard support

Hardware keyboard support

Connect your hardware keyboard and use all familiar Jupyter keyboard shortcuts.

Multitasking

Multitasking

Use Juno in Split-screen alongside other apps. Run notebooks with documentation, terminal, or other editor in front of you.

Extended on-screen keyboard

Extended on-screen keyboard

Use extended on-screen keyboard to easily insert special characters and perform quick actions on active cell.

Code completion

Code completion

Get code completion suggestions as you type, powered by your Jupyter Notebook server's kernel.

Temporary server

Temporary server

Connect to a temporary Jupyter Notebook server and run sample notebooks without any preliminary configuration.

Configuration profiles

Profiles

Add separate Jupyter Notebook server configurations for connecting to multiple remote machines.
